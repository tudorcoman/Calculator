## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This app shows a simple calculator. Two TextFields are used to receive number input from the user. Buttons were used for basic arithmetic operations. We show the result of the user's action with a Label. The user was prevented from entering a letter instead of a number. In addition, the user is prompted to enter a number by displaying an error message. Finally, a special icon has been added to the application.
